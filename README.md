# How to run
After installing all you have to do is type `onya help` you will get a prompt giving you instructions

 # Examples

 ## Printing

 ```javascript
 var x = "Hello";
 var y = " World";
 print(x,y);
 ```
 **Output:**<br>
 `>` `Hello World`

<br>

## Functions
```javascript
function DoSomething()
{
    print("Hello World!");
};

DoSomething();
```
 **Output:**<br>
 `>` `Hello World!`

 ## Functions Arguments
```javascript
function DoSomething(name)
{
    print("Hello, ", name);
};

DoSomething("John");
```
 **Output:**<br>
 `>` `Hello, John`
